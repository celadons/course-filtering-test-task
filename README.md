## Описание

### Фильтр курсов

1. В параметры функции `filterCourses` передаются два аргумента: список курсов и диапазон цен (фильтр).
2. Выполняется проверка на корректность и наличие аргументов.
3. При фильтре массива курсов применяется условие: меньшая цена фильтра меньше/равна большей цене курса и большая цена фильтра больше/равна меньшей цене курса.
4. Результат возвращается в виде списка названий курсов.

### Сортировка курсов

1. В параметры функции `sortCourses` передаются два аргумента: список курсов и направление цены (от меньшей к большей или от большей к меньшей).
2. В зависимости от направления выполняется условие и происходит сортировка.
3. Результат возвращается в виде списка названий курсов.
